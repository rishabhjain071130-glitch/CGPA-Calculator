🎓 GradePilot – GPA & CGPA Calculator

GradePilot is a modern, responsive GPA & CGPA Calculator designed to help students calculate their semester GPA, track their cumulative CGPA, and predict future graduation CGPA using a powerful What-If Predictor.

Built entirely with HTML, CSS, and Vanilla JavaScript, the project focuses on clean UI, smooth user experience, and accurate academic calculations without relying on any external frameworks.

✨ Features
📚 Semester GPA Calculator
🎯 Overall CGPA Calculator
🔮 What-If CGPA Predictor
➕ Add or Remove Unlimited Subjects
📊 Semester-wise Academic Tracking
💯 Credit-based GPA & CGPA Calculation
🌙 Dark & Light Mode
💾 Local Storage Support
📱 Fully Responsive Design
⚡ Smooth Animations & Modern Dashboard UI
📈 Progress Indicators & Statistics
📄 Export Results (PDF/CSV) (if implemented)
